Realizado por: José García Torrescusa

3.1 ¿Cuánto cuesta implantar el cloud computing?

Computación en la nube

En el sistema tradicional, las empresas necesitaban comprar servidores físicos, teniendo que adaptar las empresas sus salas para su buen funcionamiento, como la refrigeración. Pero a la hora de crecer traía muchos problemas. Por ello nació el Cloud Computing, en este modelo las empresas pagan para acceder a grupos virtuales de recursos compartidos, que se encuentran en servidores remotos. Además el servidor central administra la comunicación con los clientes.

3.2 Ventajas del cloud computing

Las empresas pueden contratar más servidores a medida que lo necesitan, disponiendo la potencia necesaria para poder satisfacer la demanda.

Las empresas alojan en la nube sus copias de seguridad de sus recursos digitales.

Las empresas pueden acceder a dus datos de forma segura y desde cualquier dispositivo, en cualquier ubicación geográfica.

La nube ofrece herramientas y plataformas para crear y probar aplicaciones, lo que reduce el tiempo de lanzamiento.

El cloud computing cuenta con recursos casi ilimitados para procesar grandes volúmenes de datos con el fin de agilizar los procesos de investigación y reducir el tiempo necesario para obtener información valiosa.

Seguridad en las comunicaciones Los servicios de cloud computing ofrecen medidas de protección frente a los ciberataques y ante la posibilidad de pérdida/borrado de datos sensibles.
